<h3>服务端和客户端环境要求</h3>
<p>服务端：需要node</p>
<p>客户端，因为用了await，请用chrome 58以上</p>
<p>运行和操作：</p>
<ul>
   <li>npm install</li>
   <li>node app.js</li>
   <li>浏览器打开 http://localhost:3000</li>
</ul>
