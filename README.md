<h3>服务端和客户端环境要求</h3>
<p>服务端：需要node</p>
<p>客户端，因为用了await，请用chrome 58以上</p>
<p>运行和操作：</p>
<ul>
   <li>npm install</li>
   <li>node app.js</li>
   <li>浏览器打开 http://localhost:8007</li>
</ul>

<h3>功能描述</h3>
<ul>
   <li>城市自主定位</li>
   <li>地址搜索和设置</li>
   <li>登陆</li>
   <li>签到</li>
   <li>注销</li>
   <li>cookie过期跳转登陆</li>
   <li>支持移动端</li>
</ul>
